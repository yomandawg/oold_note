## Box Model

* html 렌더하는데 가장 중요한 모델
* html default로 사각형
* box들이 공간을 차지하고 있다는 것과 동일한 의미

* 태그가 차지하는 경계선: **border**
* 경계들의 사이: **margin**

* 컨텐츠와 실제 박스의 사이: **padding**
* **grid** system
  * 표준12는 *약수*(1, 2, 3, 4, 6, 12)가 다양하다!
